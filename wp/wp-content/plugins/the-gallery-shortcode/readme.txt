Improve built-in [gallery] shortcode with cleaner output, optional pagination, visual exclude option and lots more.

== Description ==

Integrates with WordPress' built-in [gallery] shortcode to provide lost of unique features:

* Clean HTML5 output, with options to change on desire
* Pagination option, you can set how many items you want to have per page, you can also request for a specific portion of the gallery
* Visual "Exclude" option, you can tick the images you want to exclude from gallery! How cool is that?
* Several options on how you want your gallery images.
* Allows multiple galleries in a single post.
* & more!


== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Read the Codex on how to use <a href="http://codex.wordpress.org/Gallery_Shortcode">Gallery Shortcode</a>
4. Enjoy!
